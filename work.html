Description

This is a full stack web application of a movie application that provides a list of movies with details including the description, directors, and actors. A user can create a profile and add movies to their favorites and a to watch list.

This app uses the MERN (MongoDB, Express, React, and Node. js) stack

My role was to build:

  • ○ Frontend interface to display the movies (and movie details) and user profile setup and managements
  • ○ Backend server with the database and the API to access the database

I learned to:

  • ○ Build architected database with MongoDB
  • ○ Create a REST API with proper user documentation
  • ○ Design a Single Page Application using React
